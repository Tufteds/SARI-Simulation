# Симуляция распространения ОРВИ

## Описание проекта
Это интерактивная программа для моделирования распространения острых респираторных вирусных инфекций (ОРВИ) с использованием **агентной** и **математической** моделей. Цель проекта — изучение динамики эпидемий, влияние вакцинации, иммунитета, вспышек и мутаций на распространение инфекции.

---

## Основные возможности

- **Выбор модели:**  
  - Агентная (симуляция отдельных людей)  
  - Математическая (ODE — система дифференциальных уравнений)

- **Настройка параметров:**  
  - Размер популяции  
  - Количество дней симуляции  
  - Тип графика (линейный, столбчатый, круговой)

- **Расширенные функции:**  
  - Отсутствует

- **Визуализация:**  
  - Анимированные графики  
  - Диаграммы распределения здоровья населения

---

## Структура моделей

### Агентная модель
- Каждый человек представлен как **агент** с индивидуальными параметрами (иммунитет, статус, дни заражения).  
- Моделируются случайные контакты между людьми.  
- Учитываются инкубационный период, длительность болезни, иммунитет

### Математическая модель (ODE)
- Используется система **SEIRS** (Susceptible, Exposed, Infected, Recovered, Susceptible).  
- Дифференциальные уравнения описывают скорость перехода между группами
- Позволяет прогнозировать эпидемические кривые и анализировать параметры.

## Технологии
- **Язык:** Python 3.10+

- **Структура:** ООП, синглтоны, абстрактные классы
